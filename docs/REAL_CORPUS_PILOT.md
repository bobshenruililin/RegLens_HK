# RC3 real corpus pilot

The RC3 real corpus pilot is an internal Core 50 research workflow for testing
source handling, extraction coverage, review, and evaluation on real regulator
materials. It is not a public dataset.

## Scope

- Planned selection: `publications/pilot/core50.v1.json`.
- MCHK target: 25 internal research decisions/metadata records.
- DCHK target: 25 metadata or decision records, carrying the July 14, 2018 caveat.
- Raw documents and derived full text stay under gitignored/private storage.
- Human review is required; no automatic publication.

## Publication boundary

The public Observatory remains synthetic/demo unless a separate policy approval
changes source visibility. RC3 must not release real MCHK/DCHK judgments,
excerpts, OCR text, page text, or identifiers to GitHub Pages.

## Guardrails

- Public availability is not reuse permission.
- robots.txt is not a licence.
- MCHK remains internal-only for public release.
- DCHK publication coverage is caveated from July 14, 2018 onward and is not
  population-complete.
- No claim of complete de-identification is made for the pilot corpus.
- Student-research letters do not unlock Pages or public real releases.
- Counts describe operational pilot targets, not prevalence or regulator-wide
  rates.

# Evaluation methodology (MVP)

1. Gold corpus later: 15–20 real decisions (double annotated).
2. Milestone 1 automated gates:
   - Schema validation errors = 0 on mock outputs for fixtures
   - Evidence support rate = 100% after quote alignment
   - Provenance: every published proposition resolves to a page span
3. Pass bar for demo: synthetic fixtures fully linked on the decision page.
